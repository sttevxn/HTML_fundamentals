¿Que son los elementos de linea?

Son aquellos elementos que ocupan unicamente el mismo espacio que el contenido del mismo, a diferencia de los elementos de bloque, que ocupan el espacio completo que ocupan.

Span

Es un contenedor de linea, equivalente a la etiqueta div. Es usada frecuentemente para encerrar pequeños fragmentos de texto para darles estilo con css o localizarlos mediante JavaScript.

Ejemplo

Dentro de este texto, esta frase es azul, esta es color rojo, y esta es color verde.

q, cita de linea

Funciona de la misma manera que la etiqueta blockquote, solo que en este caso, la cita que insertemos sera reconocido por el navegador como elemento de linea.

Semantica

Dentro de la etiqueta "q" se insertará la cita que se desea. El atributo "cite" tambien esta disponible para usar si se desea.

Ejemplo

Con Cristiano hemos mantenido una competencia durante años dentro del mismo campeonato. Ha sido maravilloso y nos ha servido a los dos para seguir creciendo en nuestras respectivas carreras, pero sin estar necesariamente pendiente. Solo quería superarme para ser el mejor en general, no ser mejor que otro

Code

Se usa para mostrar codigo visualmente. Va ligada a la etiqueta pre vista anteriormente

Ejemplo

            
                .span1{
                    color: blue;
                }
                .span2{
                    color: red;
                }
                .span3{
                    color: green;
                }
